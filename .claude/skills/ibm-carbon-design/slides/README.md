# Slide template — IBM PowerPoint guidelines, rebuilt in HTML

14-slide HTML deck that mirrors the official IBM PowerPoint template: 1920×1080, IBM Plex Sans Light display type, Cyan 10 moment slides, black 1pt rules, sentence-case headlines, dash bullets, and IBM-style lower-left footers with page numbers.

## Layouts demonstrated

1. **Cover, cyan** — title + serif-italic accent.
2. **Cover, imagery half** — hero image + headline split.
3. **Contents** — numbered two-column list with hairline rules.
4. **Section divider** — Cyan 10 accent slide.
5. **Large text** — quote-scale layout at ~68pt.
6. **Data, 3 callouts** — huge display numerals with supporting text.
7. **Callout, headline** — Gray 10 moment, 2:3 text split.
8. **Boxes, 4 horizontal** — numbered tiles with 2px gutters.
9. **Text, 4 columns, dividers** — hairline-above-heading column pattern.
10. **Chart** — Carbon-palette bar chart with annotations.
11. **Table** — Carbon table with status chips.
12. **Contacts** — 4-person roster with avatars.
13. **Legal disclaimer, two columns** — verbatim IBM external/non-sales boilerplate.
14. **End slide** — black 252pt "Thank you" closer.

## Navigation

Arrow keys, Page Up/Down, Space, number keys 1–9, or the floating overlay. Print to PDF yields one slide per page.
