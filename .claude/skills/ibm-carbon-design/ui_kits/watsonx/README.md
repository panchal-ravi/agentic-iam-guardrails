# UI kit — IBM watsonx

Hi-fi recreation of the watsonx (IBM's AI & data platform) console.

## Screens

- **Home** — welcome hero, three product entry tiles (watsonx.ai / .data / .governance), foundation-models grid.
- **Projects** — data table of projects with model, asset count, status tags.
- **Prompt Lab** — full prompt editor with model selector, decoding params, Run button, live response panel. Click "Run" to simulate a response.

## Navigation

Left rail includes Home, Projects, Prompt Lab, Foundation models, Tuning Studio, Data, Governance. Other screens are stubbed placeholders.
