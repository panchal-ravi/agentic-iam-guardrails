# UI Kit — Carbon Marketing (IBM.com-style)

A single-page recreation of a typical IBM.com marketing surface built on Carbon's **Expressive** type scale.

## What's here

- **Global header** — black 48px IBM bar with primary nav + blue "Contact" CTA.
- **Hero** — Expressive 54px heading, IBM-style mixed sans/serif-italic phrasing, two CTAs, Carbon-style line pictogram.
- **Gray 100 stat strip** — large display numerals on black.
- **Product card grid** — 6 tiles with eyebrow kicker, headline, description, and a blue arrow.
- **Footer** — 4-column link grid with copyright line.

Marketing pages use the Expressive type scale, bigger hero units, occasional line pictograms, and slightly looser copy — but the palette stays gray-first with IBM Blue 60 as the single accent.
